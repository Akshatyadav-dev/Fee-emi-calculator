# 💸 Fee EMI Calculator

A simple and responsive web application to calculate EMI (Equated Monthly Installments) for course fees based on the **15/3 scheme**.

---

## 🚀 Features

* 📊 Calculates:

  * Total EMI (15 months)
  * Down payment (3 EMIs at admission)
  * Remaining 12-month EMI plan
* 📅 Select due date (5th or 10th of every month)
* 📄 Generates EMI schedule table
* 🧾 Download EMI details as **PDF (A4 size)**
* 📱 Fully responsive (works on mobile, tablet, desktop)

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Responsive Design)
* JavaScript (Vanilla JS)
* html2pdf.js (for PDF download)

## ⚙️ How It Works

1. Enter the **Total Course Fee**
2. Press **Enter**
3. View:

   * Down Payment (3 EMIs)
   * Monthly EMI (12 months)
4. Click **Download PDF** to save the schedule
